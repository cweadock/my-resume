Birthday Calculator

For my first project using Visual Studio, I created a birthday calculator with multiple function options. The user can choose from four options, as displayed in the example photo. The first options calls a function that calculates the day of the week based on the birthdate that is entered. The second option prints the next ten consecutive leap years after a year is input by the user. The third option is currently under construction and the fourth option allows the user to quit.

0hh1 Game

0hh1 is a game similar to Sudoku where a user must solve the puzzle by entering moves of X's or O's so that there is an equal number in each row and column. The game also stipulates that there cannot be three of the same move consecutively. In making this game in Visual Studio in C++, I had to focus on receiving user input, checking that is is valid, and also checking to see if the input follwed the rules of the game and fit within board dimensions.

Euchre

One of the projects that I completed in EECS 280 was creating a simulator for a game of Euchre and writing different player protocols. In this project, I gained experience with abstract data types, vectors, arrays, and polymorphism. Shown in the gif above is the "Simple" player protocol which plays strictly according to the rules of Euchre and is very basic in its strategy. The other protocol was the "Human Player" which was interactive for the user in that you could input any move and the program would respond accordingly, or, you could play against other human players.

iTunes Game

Users are able to input their favorite artists to create a "pool" of songs which the guessing game would choose the correct answer from. It would then play a preview of the track and users could guess the song, ask for a hint, pass and go to another song, or quit.
